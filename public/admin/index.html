LEVI ZIGZA Admin

Admin is local-only

GitHub Pages is static — there is no upload API here. Run the portfolio locally to sign in and manage chamber media.

  1. Clone the repo and copy .env.example → .env (set ADMIN_PASSWORD).
  2. Run npm install then npm run dev (Vite + Express).
  3. Open http://localhost:5173/Levi-Zigza-Artist-Portfolio/admin/

Vite proxies /api to the Express host on port 5174.

Chamber upload

Sign in with ADMIN_PASSWORD from your .env.

Upload media

Files land in public/uploads/ and appear in chamber pages via the manifest.
